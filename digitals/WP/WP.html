Web Programming & Scripting Languages

Scripting Language

Scripts are programs just like any other programming language. They can execute on the client-side or on the server-side. Scripting languages do not require a compilation step; instead, they are interpreted.

1. Client-Side Scripting Language

Client-side scripts are processed on the browser or local computer. They allow changing the contents of the web page dynamically.

Functions:
  1. Interact with temporary storage.
  2. Make interactive web pages.
  3. Interact with local storage.
  4. Send requests for data to the server.
  5. Send requests to the server properly.
  6. Work as an interface between the server and users.

2. Server-Side Scripting Language

It is a script that runs on a server, dealing with the generation of web page content.

Functions:
  1. Querying the database.
  2. Operations over the database.
  3. Access/Write a file on the server.
  4. Interact with other servers.
  5. Structuring web applications.
  6. Processing user input.

HTML (HyperText Markup Language)

HTML is the standard markup language for creating web pages. It provides the structural foundation of websites by defining elements such as headings, paragraphs, links, images, forms, and multimedia content.

Key Characteristics:
  • Markup Language
  • Platform Independent
  • Foundation of the Web
  • Static Content

Why is HTML Essential Before PHP?

  1. Output Layer: PHP often outputs HTML.
  2. Form Integration: PHP processes HTML forms.
  3. Template Structure: HTML provides the skeleton.
  4. Client-Server Model: HTML acts as the client-side interface.

HTML Elements and Tags

HTML tags are keywords enclosed in angle brackets < >.

  • Paired Tags (Container Tags): Come in pairs. E.g., <p> ... </p>
  • Self-Closing Tags (Empty Tags): No closing tag. E.g., <img>, <br>, <hr>

Block-level vs Inline Elements

Type Description Examples
Block-level Elements Always start on a new line and take up the full width available. <div>, <p>, <table>, <h1>
Inline Elements Do not start on a new line and only take up as much width as necessary. <span>, <a>, <strong>, <img>

Text Content and Examples

Headings:

HTML Forms

Forms are the primary method for collecting user input on websites. In PHP, this input is processed on the server.

<form action="process.php" method="POST">
    <!-- Form elements go here -->
</form>

Form Attributes

  • action: URL where form data is sent.
  • method: HTTP method (GET or POST).
  • name : form identifer
  • target:where to display respone
  • enctype: Encoding type (Essential for file uploads: multipart/form-data).

What is PHP

The PHP(Hyper Test Preprocessir) is a server-side scripting language that allows web developers to create a dynamic content that interact with databases

PHP is a server-side scripting language embedded in HTML

It is integrated with a number of popular databases including MySQL,PostgreSQL,Oracle,sybase,informix and Microsoft SQL server

Basic Syntax

PHP scripts start with <?php and end with ?>. Files use the .php extension.

<?php
    echo "Hello World";
    // PHP statements end with a semicolon
?>

Variables

PHP is loosely typed,which means we dont have to explicitly state the tpye of variable

Variables start with $followed by the variable name

A variable can not start with Number

<?php
    $txt = "Rohan";
    $x = 5;
    echo $x; 
?>

Code Examples for Scope:

Local Scope:

function myFunction() {
    $x = 10; // Local
    echo $x;
}

Global Scope (using global keyword):

$x = 5; $y = 5;
function add() {
    global $x, $y;
    return $x + $y;
}

Static Scope:

function myTest() {
    static $x = 0; // Value persists
    echo $x;
    $x++;
}
